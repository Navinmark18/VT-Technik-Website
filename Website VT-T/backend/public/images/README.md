# Bilder für die Website

Lege hier deine Bilder rein, und sie werden automatisch auf der Website verwendet.

## Dateinamen:
- `lichttechnik.jpg` oder `lichttechnik.png` - für Lichttechnik-Karte
- `videotechnik.jpg` oder `videotechnik.png` - für Videotechnik-Karte

## Beispiele:
```
/images/lichttechnik.jpg
/images/videotechnik.jpg
```

Die Website prüft automatisch, ob diese Dateien existieren und verwendet sie.
